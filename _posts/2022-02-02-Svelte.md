### 1. degit 설치

```bash
$ npm install -g degit
```

### 2. Svelte 템플릿 내려받기

```bash
$ npx degit sveltejs/template svelte-app
```

### 3. 모듈 설치

```bash
$ cd svelte-app
$ npm install
```

### 4. 실행

```bash
$ npm run dev
```
