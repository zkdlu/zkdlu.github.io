# Set

순서가 없는 컬렉션

## HashSet

사용 예제

```java
public static void main(String[] args) {
    HashSet<String> set = new HashSet<String>();
    set.add("Hello");
    set.add("World");
    set.add("Hello");
    System.out.println(set);
}
```

Iterator 사용 예제

```java
public static void main(String[] args) {
    HashSet<String> set = new HashSet<String>();
    set.add("Hello");
    set.add("World");
    
    Iterator<String> iter = set.iterator();
    while (iter.hasNext()) {
        System.out.println(iter.next());
    }
}
```
